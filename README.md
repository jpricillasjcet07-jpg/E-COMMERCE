# E-COMMERCE# 🛒 E-COMMERCE

A full-stack E-Commerce web application developed using React, Spring Boot, and MySQL.  
This project provides an online shopping experience with user authentication and product management.

## 🚀 Features

- User Registration and Login
- JWT Authentication
- Product Listing
- Product Search
- Add to Cart
- Wishlist
- Responsive User Interface
- REST API Integration

## 🛠️ Technologies Used

### Frontend
- React.js
- JavaScript
- HTML5
- CSS3
- Axios
- React Router

### Backend
- Java
- Spring Boot
- Spring Data JPA
- REST API

### Database
- MySQL
